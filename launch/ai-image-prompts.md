## Tips
- In ChatGPT: paste the prompt exactly, it will generate the image
- In Canva Magic Media: paste prompt, choose "Photo" style
- In Leonardo.ai: paste prompt, set Width=1080 Height=1920, use "Leonardo Diffusion XL" model
- After generating: add CampusClip logo + caption text in Canva before posting
